Para usar el app shiny, se requiere tener guardado en memoria el dataframe autos_data creado por el rmd, ya limpiado y con los NA imputados. En el app aparecen tres pestañas, uno para análisis univariado, uno para análisis bivariado, y el último para multivariado. En la pestaña de univariados, el primer dropdown list selecciona cual variable se quiere visualizar, mostrandose una gráfica de barras en cada caso, siendo los bins las categorías para variables categóricas, y rangos de valor para las numéricas. En la pestaña de bivariados, los dos dropdown lists se usan para elegir el primer y segundo variable sobre las que se quiere hacer un análisis. En caso de ser las dos númericas se mostrará un scatterplot, si una es numérica y la otra categórica, se mostrará un box plot con una caja para cada variable categórica, y en caso de ser ambas categóricas se mostrara una grafica de barras donde cada barra corresponde a las categorías del primer variable, y sus divisiones a las del segundo variable. En la última pestaña, de multivariada, se mostrará un plot de los datos graficados según las dos componentes principales mayores, y una segunda gráfica que es de barras muestra la división de la varianza entre las componentes principales. Al escojer una variable en el primer drop down list, los puntos graficados en la primera gráfica serán coloreados de acuerdo con a qué categoría pertenecen (en caso de variable categórica), o según un gradiente de menor a mayor valor (en caso de variable numérica).
